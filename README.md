# Project Title
# Trivia Challenge

## Project Description

The purpose of the project is to ask user a series of question and store the answer and display the results with the score obtained by the user in the java. Basically, it ask the user the question and user need to answer the questions.


### Dependencies
To run the code we need to have java installed on the System. We need to have IDE as well but we can write the code in the notepad.

### How to run the project
If they have things installed, how can they run your project?
Let's say user have installed the VS Code then user needs to open the project in their's IDE and click on the Run button which will be on the top right of the project.

### How to play the game

First, program will ask the input about name. User will enter the name and again program will ask the wide of the catacomb user need to enter the size between 5-10 else the program will show the not a valid size. Again user need to answer the program's question which direction they want to move either east,west,north and south. At last user needs to escaped the catacomb.
